Help partners submit their data in the right format

No more clogs

  • Ingest csv files from non-technical team members
  • Spec your format by providing examples
  • Check the plumber's work and make adjustments with a few clicks
  • Get a convenient URL to share so your colleagues can check their data with immediate feedback
  • Use the same parser in production and let the plumber take charge!

How it works

  1. Load your CSV file.
  2. Let Dataplumber.app guess data types for each column.
  3. Customize your column parsers using Python.
  4. Generate a python parser and URL for others to validate their data.

Features

  • Data never leaves your browser
  • Automatic data type detection
  • Supplies generated parser code
  • Custom python column parsers for those who know how to code
  • Data validation against an existing format via sharable URL
  • Open source codebase
