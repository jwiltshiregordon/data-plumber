How it works

  1. Load your CSV file.
  2. Let Dataplumber.app guess data types for each column.
  3. Customize your column parsers using Python.
  4. Generate a URL for others to validate their data.

Features

  • Data never leaves your browser
  • Automatic data type detection
  • Supplies generated parser code
  • Custom python column parsers for those who know how to code
  • Data validation against an existing format via sharable URL
  • Open source codebase (github)
